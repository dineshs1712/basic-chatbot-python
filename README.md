🤖 Simple Python Chatbot

📌 Overview

This project is a simple rule-based chatbot built using Python.
It responds to user inputs based on predefined data stored in a JSON file and provides random replies for a more natural conversation experience.

---

🚀 Features

- 💬 Responds to predefined user inputs
- 🔀 Generates random responses
- 📁 Uses JSON for flexible data storage
- 🧠 Beginner-friendly logic

---

🛠️ Technologies Used

- Python
- JSON

---

📂 Project Structure

chatbot-project/
│
├── mini_chatbot.py      # Main chatbot logic
├── datas.json      # Chatbot response data
└── README.md       # Project documentation

---

▶️ How to Run

1. Clone the repository:

git clone https://github.com/dineshs1712/basic-chatbot-python.git

2. Navigate to the project folder:

cd basic-chatbot-python

3. Run the chatbot:

python mini_chatbot.py

💡 Example Interaction

YOU: hello

YOU: how are you
BOT: I'm doing great!

YOU: exit
